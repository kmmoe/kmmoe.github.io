请插入AutoCAD光盘

发表于 2016-02-01 更新于 2023-06-15 分类于 软件研究院

从网上下载的都是exe安装文件，有的iso都是民间自制的。关于exe打包为iso不再介绍了。 [nottomatter] 1.准备工具 UltraISO和CAD安装镜像。 2.用UltraISO打开CAD安装镜像，找到setup.ini文件，将它拖到你的硬盘上，打开，在第二行加入：

[SETUP]

USE_REMOVABLE_DRIVE=YES

3.保存，拖至镜像覆盖原文件，保存镜像即可。 Tips:2008不支持，2012/2014/2015/2016测试通过。

Wordpress翻译

发表于 2016-01-02 更新于 2023-06-15 分类于 码农 ， WordPress

© Sunplace,2016 1. 后台为显示为英文的处理方法 找到根目录下的wp-config.php，在末尾加入：

define(‘WPLANG’, ‘zh_CN’);

随后进入后台，点击“更新”，下载安装包就好。 2. 网站显示英文的处理方法 找到wp-content/themes/你的主题名/languages里面的mo和po文件都是翻译文件。 2.1 下载任意一个po文件，使用Poeditord打开，修改译文后，保存。之后点击“文件”-“编译为MO…”保存为zh-CN.mo。 2.2 如果只有pot文件，请点击下方的“创建新的翻译”： 然后在要翻译的语言里填入zh-CN 3.自定义添加翻译 如果完成了上面几步的还是英文，那么就把整个主题目录下载到本地，利用浏览器“急速模式”或者是Google Chrome对英文位置右键，选择“审查元素”，配合Dreamweaver找到所在页面。 使用记事本打开zh-CN.po，在末尾加上：

#: [文件路径]:[待翻译的英文所在行号]
msgid “[原文]“
msgstr “[翻译后的文字]“

例子：

#: searchform.php:11
msgid “Search &hellip;”
msgstr “ “

Office部署工具（OCT）

发表于 2015-12-31 更新于 2023-06-15 分类于 软件研究院

© Sunplace,2015

参考：

  1. 使用 OCT 自定义 Office 2013
  2. Microsoft Office 2019早期预览版详细图文安装教程

  1. 查看手中的Office镜像中是否包含Admin文件夹，有为批量授权版，没有为零售版。

  2. 没有Admin文件夹的可以到以上的链接去下载，注意下载的工具要与镜像架构和版本对应（例如32位的Office 2013）。

  3. 部署支持Office 2007及以上的版本。

  4. 运行自定义安装工具，cmd进入光盘，运行以下命令：

    setup.exe /admin

    启动Office自定义工具，在工具内设置安装选项，之后保存到某个文件夹，本文将update.msp保存到某个文件夹下的updates文件夹内。

  5. 指定首次安装调用的部署文件（MSP）的位置：

    setup.exe /adminfile /updates

    这个命令只对本机创建的MSP文件有效。
