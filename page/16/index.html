ACDSee Pro 8 激活

发表于 2015-07-15 更新于 2023-06-15 分类于 软件研究院

2015-07-15 © Sunplace,2015

准备工作： 下载英文版和简体中文版离线安装包（见附件）。 下载注册机ACDSee Pro v8.1.Keymaker-CORE v4.2.exe（见附件）。 1. 修改host文件（C:\Windows\System32\drivers\etc），最后加上： 127.0.0.1 acdid.acdsystems.com 然后保存。 2. 安装英文版，进入注册表，将 HKEY_LOCAL_MACHINE\SOFTWARE\ACD Systems\ACDSee Pro\80\LClient导出为reg文件，备用。 3. 卸载英文版，安装简体中文版。 4. 双击之前导出的reg文件。 5. 断网。启动注册机，点击Gen LicKey生成一枚License Key。（注意：不要关闭注册机） 6.启动ACDSee，将 License Key粘贴到输入您的许可证密钥，然后继续。 7. 点击稍后注册。 8. 软件会自动重启，之后帮助-离线注册。 9. 将您的请求码粘贴到注册机的Request Code中，点击Gen Unlock，复制生成的Application Unlock Code，关闭注册机。 10.点击输入解锁码，粘贴 Application Unlock Code，点击验证。 11.成功。

使用手机来代替电脑摄像头

发表于 2015-04-05 更新于 2023-06-15 分类于 安卓

2015-04-05 © Sunplace,2015

 
引言 爷爷最近要进行养老保险的定期人脸认证，就是“刷脸”。但是系统不支持笔记本的内置摄像头。 前提： 1.一台带摄像头的安卓手机。 2.以下环境满足任意一项：WLAN（无线局域网）、蓝牙、USB数据线。 步骤： 1.下载DroidCam的PC版本和手机版本，推荐去官网下载。 2.电脑手机分别安装。 3.本文用WLAN的连接方式。打开手机端，会显示相关的信息，主要是IP地址。 4.把IP地址填进电脑端的Deivice IP内，可选Video（视频传输）、Audio（音频传输）、Veideo Quality（视频质量），之后点击Start（开始）就可以使用了。

Adobe Creative Cloud安装技巧

发表于 2015-03-08 更新于 2023-06-15 分类于 软件研究院

2015-03-08 © Sunplace,2015

 
引言 最近发现，在Win8.1下安装AcrobatXI会有问题，安装别的Adobe就没问题…… 安装Acrobat XI Pro 1.下载安装Adobe Creative Cloud，之前叫做Adobe Application Manager。 2.注册一个Adobe ID，之后在首选项中设置一下软件安装位置。 3.点击试用即可安装AcrobatXI。 查找下载的文件 1.如果想保留下载的文件，首先我们要进入C:\Program Files (x86)\Adobe\Adobe Creative Cloud\Utils\zip 2.将7za.exe重命名为7za.exe_。 3.回到Adobe Creative Cloud面板，下载所需软件。 4.下载完成后会提示安装失败，这是因为没有解压。 5.win+r进入%temp%，找到一个类似{341EFC8C-237F-4D9C-BC8A-174CFA44432B}的文件夹，里面就是未解压的安装文件了。 6.将安装文件备份出来，之后把7za.exe_改回7za.exe，在面板中点击重试即可继续安装。

tumblr和WordPress

发表于 2015-02-26 更新于 2023-06-15 分类于 网站 ， WordPress

2015-02-26 © Sunplace,2015

 
由于之前我在万网免费得到一个云主机，于是就用来搭建了BLOG，接触php也没几天，看上了WordPress，算是慕名的吧。 WordPress和tumblr几乎没区别，硬要罗列的话，有： tumblr优点：1. 对于小白，轻松搭建。2.维护方便，后台功能全面。3.服务器不在国内。 WordPress优点：1.容易定制。2.多语言。3.主题多，插件多。4.前端处理更丰富。 以上反之就是各自的缺点。 虽然我搭建有自己的WordPress，但技术方面的文章还是留在tumblr，主要担心被墙，听说tumblr打算进军中国，那只能对中国推出阉割版，同时墙掉国外版。至于为什么，当然是因为tumblr里面的NSFW。 为什么不用国外的主机？因为tumblr是国外的主机，还免费提供一个二级域名。但是WordPress自带的域名是被墙的，想要访问，只好上国内提供的域名（不一定是.cn），如果有所谓不良言论，直接XXXX。

将用户文件夹移动到非系统盘

发表于 2014-12-01 更新于 2023-06-15 分类于 OS

2014-12-01 © Sunplace,2014

 
1.安装Windows，直到出现这个界面。 2.按下”Shift”+“F10”，调出命令行，输入”regedit”调出注册表。 3.找到“\HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVeriosion\ProfileList”。 4.在右侧找到”ProfilesDirectory”，将数值数据改为“x:\Users”（x为系统盘外的盘符）。 Tips：x盘符必须事先分配好，如果还没有分配盘符，可以进入安全模式分配。
