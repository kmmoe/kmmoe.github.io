如何不借助下载工具下载在线流媒体

发表于 2009-12-16 更新于 2023-06-15 分类于 网站

2009-12-16 © Sunplace,2009

最近在看11eyes，所以没有更新。不过对于在线播放的媒体格式（.flv,.wma,.mp3,.swf这类的。）可以在它们缓冲完后，在你的电脑里面发现它们！ 方法是找到如下位置： 打开你的浏览器：【工具】→【IE（Internet）选项】→【常规】→浏览历史记录→【设置】 一般位置（Windows 7）C:\Users\当前登录用户名\AppData\Local\Microsoft\Windows\Temporary Internet Files （Windows XP）C:\Documents and Settings\当前登录用户名\Local Settings\Temporary Internet Files 里面就是你上网的全部缓存了，当你听完一首在线的音乐（通常是Windows Media Player播放的。）或者欣赏完一段视频（我看的是56，不知道别的可不可以。），就能在这个文件夹下找到它们。 我在Windows XP操作时，音频、视频可以直接拖到桌面上。在Windows7操作时，音频不能复制到其他地方，只要你查看文件得属性，复制上面的网址，就可以了。用Windows Media Player可以直接打开URL播放，然后在播放的时候，【文件】→【另存为】就可以了。也可以用迅雷下载。 哈哈，大功告成！

Adobe CS的破解（适用于CS4+）

发表于 2009-10-31 更新于 2023-06-15 分类于 软件研究院

2009-10-31 © Sunplace,2009

1. 确保之前的版本卸载干净（当然笔者目前了解5.5和6可以共存）。 2.关于Adobe Acrobat的事先说明： 如果你安装的Adobe Acrobat需要在安装时输入序列号（没用试用的按钮），拿就去度娘一个输进去。 3.事先准备好破解文件amtlib.dll(×86，×64如果你安装的是64位的)。 4.断网。 5.安装Adobe CS，这个时候就没有了登录的按钮，选中试用，安装……（安装的时候可以适当取消勾选一些不需要的组件） 6.安装完成后，别急着联网。先把每个软件都运行一遍。（可解决Extended版变回普通版的问题） 7.覆盖目录下的amtlib.dll文件。（Adobe Acrobat也需要）。 ===============================可精简项目（视个人情况）========================================== .8.1 直接卸载 Adobe Help Adobe AIR Adobe Web Brower（大概是这个） （如果你对这些软件熟悉，还能继续卸载。） 8.2 删除（5 5.5 6通用，不能删除请注销后再删。删除之前确保你不需要这个软件。） Adobe Device Central CS5 安装目录:\ AAMUpdaterInventory C:\Program Files\Common Files\Adobe Adobe OnLocation CS5 C:\Program Files\Common Files\Adobe CS5ServiceManager C:\Program Files\Common Files\Adobe SwitchBoard C:\Program Files\Common Files\Adobe ARM C:\Program Files\Common Files\Adobe UWA C:\Program Files\Common Files\Adobe\OOBE\PDApp\ ======================================================================================== 9.联网之前进入 C:\Windows\System32\drivers\etc\hosts 加上 127.0.0.1 activate.adobe.com 127.0.0.1 practivate.adobe.com 127.0.0.1 ereg.adobe.com 127.0.0.1 activate.wip3.adobe.com 127.0.0.1 wip3.adobe.com 127.0.0.1 3dns-3.adobe.com 127.0.0.1 3dns-2.adobe.com 127.0.0.1 adobe-dns.adobe.com 127.0.0.1 adobe-dns-2.adobe.com 127.0.0.1 adobe-dns-3.adobe.com 127.0.0.1 ereg.wip3.adobe.com 127.0.0.1 activate-sea.adobe.com 127.0.0.1 wwis-dubc1-vip60.adobe.com 127.0.0.1 activate-sjc0.adobe.com 10.联网，破解成功。

Office 官方卸载工具Fix it（Win）

发表于 2009-07-12 更新于 2023-06-15 分类于 软件研究院

2009-07-12 © Sunplace,2009

引言 有的童鞋无法安装Office的原因可能是之前安装的版本没有卸载干净。 1. 登陆Microsoft的solutions https://support.microsoft.com/kb/971179/zh-cn 2. 下载相应的Fix it 原则上是电脑里装了几个版本就下几个Fix it，下载后的运行顺序从旧到新。 3. 运行之后，理论上就可以解决卸载的问题了。
