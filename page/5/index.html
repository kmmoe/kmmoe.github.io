使用第三方recovery来移除锁屏密码

发表于 2020-03-04 更新于 2023-06-15 分类于 安卓 ， 软件研究院 ， 码农

© Sunplace,2020 复工了，工作手机却忘了锁屏密码。网上的都是安卓老版本额移除方法，没有找到Android9的，于是就冒死一试。 前提条件

  • Android9+（本机为9.0）
  • 安装有第三方recovery（本机为OrangeFox Rec）
  • 电脑有adb（可选）

1.手机开机，连接电脑用adb reboot recovery手机关机，电源+音量上====>两种方法进入recovery 2.点击Menu 3.点击More… 4.点击Terminal 5.在终端（Terminal）中输入以下命令

# 进入文件夹
~ # cd /data/system

列出该文件夹里的所有内容，如果有lock开头的文件就是锁屏密码

/data/system #ls
appops.xml
…
watchlist_settings.xml

删除lock开头所有的文件

/data/system # rm lock*

6.返回到Menu界面，点击Reboot 7.点击System，等待重启完成之后，就能把锁屏密码移除，但没有移除指纹。

ZreoTier内网穿透

发表于 2020-02-17 更新于 2023-06-15 分类于 网站 ， 软件研究院

© Sunplace,2020 一切的一切，还是要从那只蝙蝠说起…… [disdain] ZreoTier能让我们像操作局域网电脑（不仅限于电脑）一样操作任意一台能接入互联网的电脑。如果电脑有个外网IP，就不需要以下操作了。 1.访问https://www.zerotier.com/，注册一个账户。 2.登录后台https://my.zerotier.com/network/，点击Create a Network。 3.之后点击这个刚创建的网络（Network）。 4.下拉到Access Control（访问控制选项），选PRIVATE表示加入此网络的成员（计算机）必须在后台审核通过之后才能使用，选PUBLIC表示知道这个Network的ID就能加入（不需要审核放行）。 5.在准备加入此网络的电脑里安装ZeroTier，下载地址https://www.zerotier.com/download/ 6.安装完成后运行，点击任务栏上的图标选择Join Network…（加入网络） 填好之前创建的Network的ID，点击Join。

还可以选择开机运行 7.把所有需要“组网”的电脑都Join之后，在后台Managed IPs里能看到10.241.37.69这个就是在此“局域网”里的IP地址了。 访问的时候需要双方都开启此软件，如果在步骤4选择的是PRIVATE，那么还需要在Members里把Auth打上钩来允许加入。

我们现在拿到了这个IP，可以做任何两台局域网能做的事情。
