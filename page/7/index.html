Windows 10使用smb共享

发表于 2018-08-20 更新于 2023-06-15 分类于 OS ， 软件研究院

© Sunplace,2018 1.在文件资源管理器的地址栏上粘贴 控制面板\程序\程序和功能 启用或关闭Windows功能 勾选如图所示的组件，然后点击确定。 2.在文件资源管理器的地址栏上粘贴 控制面板\网络和 Internet\网络和共享中心\高级共享设置 专用： 来宾或公用： 所有网络： 最后一项启用秘密保护共享和关闭密码保护共享都可以。 3.在共享的文件夹右键属性： 共享-高级共享-勾选 共享此文件夹-设置共享名（可选）-权限-添加: 输入对象名称来选择-本例是_Administrator_（要确保该用户启用，设有密码）-检查名称： 之后会显示_你的主机名/Administrator_，点击确定。 回到之前的窗口，选中_Administrator_，在下方选中完全控制，然后确定： 回到窗口高级共享，点确定，关闭。 4.在播放器或者智能电视、机顶盒上选择连接方式为SMB（或Windows共享），IP地址为电脑的IP，用户名和密码就是_Administrator_和他的登陆密码。

在Windows 10 家庭版中启用组策略

发表于 2018-06-08 更新于 2023-06-15 分类于 OS ， 软件研究院

将以下内容保存为bat或cmd文件，右键以管理员身份运行即可：

@echo off
pushd “%dp0”
dir /b C:\Windows\servicing\Packages\Microsoft-Windows-GroupPolicy-ClientExtensions-Package3*.mum >List.txt
dir /b C:\Windows\servicing\Packages\Microsoft-Windows-GroupPolicy-ClientTools-Package~3*.mum >>List.txt
for /f %%i in (‘findstr /i . List.txt 2^>nul’) do dism /online /norestart /add-package:”C:\Windows\servicing\Packages\%%i”
pause

等待配置完成，就能正常使用组策略了。

安卓wifi去感叹号与vpn冲突

发表于 2018-04-13 更新于 2023-06-15 分类于 安卓

© Sunplace,2018 升级到7.0、7.1、8.0、8.1的安卓，由于总众所周知 [disdain] 的原因，wifi图标会出现叉叉。 这个在7是没有什么问题的，一条命令就能解决，和VPN也没有冲突。

adb shell “settings put global captive_portal_https_url https://captive.v2ex.co/generate\_204“;

但是在8.0+的情况是：如果有叉叉，就不能上网。如果去掉了叉叉，VPN能连上，但是不能联网。 不过，把captive_portal_https_url改为默认值，就能通过VPN联网。折中的办法不是重复打以上的命令，用一款能设置captive_portal_https_url的app。 想用VPN的时候，设为默认（空值），不想用VPN的时候切换到其他服务器。
