修复EFI启动

发表于 2018-02-02 更新于 2023-06-15 分类于 OS ， 码农

适用于Windows8及以上内核的pe系统。

  1. 使用DiskGenius将系统盘（通常是C盘）中的EFI文件夹中的Boot和Microsoft复制到ESP分区。

  2. 为ESP分区分配盘符。

  3. 在命令行中键入以下命令：

    bcdboot c:\Windows /s g: /f uefi /l zh-cn

语法：bcdboot [Windows文件路径] /s [ESP挂载的盘符] /f uefi /l zh-cn

下载地址：https://github.com/Wind4/vlmcsd/releases

搭建Windows的服务器

  1. 复制Windows/intel/vlmcsd-Windows-x64.exe到C:\Windows\System32\vlmcsd.exe

  2. 依次运行以下命令：

    # Create System Service
    sc create KMSSrv binpath= %windir%\System32\vlmcsd.exe type= own start= auto displayname= “KMS Emulator”

    Run Vlmcsd

    net start KMSSrv

    Open Port:1688

    netsh advfirewall firewall add rule name=”KMS Emulator Server” dir=in action=allow protocol=TCP localport=1688

以下为附加： 《微软产品KMS激活指南》

Windows激活命令:

删除指定密钥：

slmgr /upk XXXX-XXXX-XXXX-XXXX-XXXX

删除所有密钥：

slmgr /upk

安装密钥：

slmgr /ipk XXXX-XXXX-XXXX-XXXX-XXXX

设置激活服务器地址：

slmgr /skms 服务器IP地址:1688

激活Windows：

slmgr /ato

验证有效期：

slmgr /xpr

Office 2010+激活命令：

切换到Office安装路径 \Microsoft Office\OfficeXX(2016为Office16) 删除指定密钥（密钥最后5位）：

cscript ospp.vbs /unpkey:XXXXX

删除所有密钥：

cscript ospp.vbs /unpkey

安装密钥：

cscript ospp.vbs /inpkey:XXXXX-XXXXX-XXXXX-XXXXX-XXXXX

设置激活服务器地址：

cscript ospp.vbs /sethst:服务器IP地址
cscript ospp.vbs /setprt:1688

激活Office：

cscript ospp.vbs /act

验证有效期：

cscript ospp.vbs /dstatus

零售版转VOL版*：

cscript ospp.vbs /rearm
