虚拟机体验Mac OS X

发表于 2011-02-09 更新于 2023-06-15 分类于 OS

2011-02-09 © Sunplace,2011

引言 从网上下载的原版Mac系统镜像，虽说是原版，但也没见多少破解成功。笔者也是试着去破解，当然虚拟机和PC都试过了。最好的也是进到了“灰屏”+“风火轮”的界面就没有下文了。苹果可是不好黑的，笔者花了将近一个月也不能成功。 既然不能破解原版的镜像，那么就从网上下已破解的镜像。 1-3点是带读者了解Leopard的安装过程。 具体可以跳过直接看4。 【提示】此文是针对Mac OS X 10.5（Leopard花豹）而不是Mac OS X 10.6（Snow Leopard雪豹）的非官方教程。 1.计算机配置（hp4416s） 处理器：AMD Athlon（速龙） II 双核 M300 主板：惠普 7013 内存：2GB（尔必达 DDR2 800MHz / 金士顿 DDR2 800MHz） 主硬盘：日立 HTS723225L9A360（250GB） 显卡：ATI Mobility Radeon HD 4330（M92）（512MB /惠普） 显示器：奇美 CM01444（14英寸） 系统：Windows 7 家庭高级版（其实这个系统倒是无所谓的，毕竟现在是用虚拟机） 【提示】黑苹果“未遂”的主要原因是无法破解硬件驱动。 2.准备工具 Leopard-10.5.2-AMD-EFI.iso（经AMD内核破解的Mac OS X） md5: 6435FAE8F1E0106291163987513B32AA VMware Workstation 7.1 3.实施过程 3.1新建虚拟机，在选择操作系统时选“Free 64-bit”。CD/DVD就选Leopard-10.5.2-AMD-EFI.iso。其他配置默认，全IDE模式（硬盘+DVD）。 【提示】至于在别的配置是否能正常运行，不得而知。但是系统选择却不能选“Windows”，不支持从Windows自启动。 3.2安装Mac OS X 3.2.1选择你看的懂的安装语言，按照安装提示安装。 3.2.2一切都顺利进行……（P.S.那还需要我干嘛？！）在选择安装在硬盘位置时，对于首次安装Mac的读者来说，是看不到任何盘的…… 3.2.3此时，在屏幕最上方可以看到可爱的菜单。选择“实用工具”-“磁盘工具…”。如下图（点击查看大图）： 图1：启动界面 图2：选择安装语言 图3：欢迎界面 图4：硬盘未被格式化 图5：格式化硬盘 图6：选择安装硬盘 图7：点击左下角的“自定”按钮，选择自己需要安装的选项 图8：安装进行中 图9：进入界面 【提示】Darwin和EFI是集成在此iso镜像中，然而完成全部安装后无法引导进入Mac。此时，作为笔者的我，在原先安装的Windows XP上，安装Tinkboot，之后将安装后的Mac OS硬盘接到虚拟机Windows XP上，顺利进入Mac OS X。 图10：安装完成，进入系统发现鼠标用不了了…… 【提示】Leopard-10.5.2-AMD-EFI.iso未在实机上进行过操作，可能未破解硬盘（OSInstall）文件，请事先备份硬盘数据！（不仅仅是C盘） =============================================================================================== 2011-4-13 18:03:53重新编辑 4.准备工具 Mac.OS.X.Leopard.[Intel.AMD].rar（经AMD内核破解的Mac OS X） md5: A05F665FFEF562E7777EC86C603B8384 下载页面： 《虚拟机体验苹果系统》(Mac OS X Leopard On VMware.[支持Intel和AMD的CPU].更新安装视频)[压缩包] VMware Workstation 7.1 5.安装 5.1安装虚拟机VMware Workstation 5.1解压下载后的Mac.OS.X.Leopard.[Intel.AMD].rar，如图12所示。双击“FreeBSD.vmx”文件，或者用虚拟机打开。 图11：\Mac OS X Leopard\Mac OS X Leopard 内的文件(可能跟读者的有些出入，这是因为我运行后再截的图) 5.2在启动虚拟机前，进行CPU的微调，如图12。 图12：选择左边的Processors，右边设置为此图的状态 5.3启动虚拟机，虚拟机提示，选“copy it”和“move it”都可以。之后会有图13的界面。 这样就大功告成了，作者很用心的帮我们安装了一些常用软件。之后，享受Mac OS X吧。鼠标当然没有问题咯。
