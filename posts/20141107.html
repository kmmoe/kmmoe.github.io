Windows手动搭建php环境

发表于 2014-11-07 更新于 2023-06-15 分类于 网站 ， 软件研究院 ， 码农

2014-11-07 © Sunplace,2014

0. 准备工作： 下载以下3个文件： PHP（zip） MySQL（完整版约287M的msi） Navicat for MySQL ============ PHP配置 ========== 1. 解压文件，重命名为php，放入C盘根目录，修改php.ini-development为php.ini并打开，修改php.ini（Ctrl+F搜索等号前面的内容）：

short_open_tag = On

extension_dir = “C:/php/ext”

cgi.force_redirect = 0

cgi.fix_pathinfo=1

fastcgi.impersonate = 1

extension=php_curl.dll

extension=php_gd2.dll

extension=php_mbstring.dll

extension=php_mysql.dll

extension=php_mysqli.dll

extension=php_pdo_mysql.dll

extension=php_pdo_odbc.dll

extension=php_xmlrpc.dll

date.timezone =Asia/Shanghai

2. 运行php-cgi.exe如果没有任何字符，证明配置基本正确。 如果出现缺少dll，安装C++运行库解决。 ============ MySQL配置 ============ 3. 安装MySQL，选择Custom，只安装MySQL Server即可，注意安装默认数据库位置为：C:\ProgramData\MySQL\MySQL Server 5.6\data 3.1 展开到MySQL Server选择×64或×86，点击绿色的向右箭头。 3.2 在右侧点一下MySQL Server，在下方就会出现一个”Advanced Options”。 3.3 点击”Advanced Options”就能自定义数据库的安装位置和数据文件的存储位置了。 3.4 点击”Execute”。提示”Ready to download”表示更新MySQL Server的版本。 设置密码（连接数据库密码）： ============ Navicat for MySQL配置 ============ 4. 运行navicat.exe，点击连接，连接名称自定义，密码输入安装MySQL所设的密码： 数据库的文件所放置的位置为MySQL安装时设置的位置。 ============ IIS配置 ============ Windows 7可用PHP Manager 1.2 for IIS 7 5. 从控制面板中添加你的IIS，确认CGI启用 6. 打开IIS，在左侧选择你的主机名，右侧为默认文档添加index.php,处理程序映射，添加脚本映射。 7. 点击是。 ============ 完成配置 ============ 8. 添加一个网站，在文件夹下编写一个index.php文件：
